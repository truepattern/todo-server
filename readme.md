# Todo Server
This is a simple todo api server based on node.js, express and mongoose

# Directory structure
 * apps - all modules go here
  * todos - todo module 
 * test - all unit tests go here
 * server.js - main server

## check it live
 * [hosted on heroku](http://aonx-todo.herokuapp.com)
